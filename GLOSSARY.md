# 바이스슈발츠 용어 글로서리 (JP → KR)

룰북·Q&A·에라타 한글화의 **단일 용어 기준**. 표기는 인접 프로젝트 **`ws_auto`(전 166작품·기본카드 41,629종 전량 한글화)** 코퍼스의 표준 표기에 맞춘다 — 룰북과 카드 텍스트가 같은 용어를 쓰게 하기 위함.

**표기 방침**: 카타카나 외래어는 음차(국립국어원 외래어 표기법 준수), 한자 게임 용어는 정착된 한국어, 색은 고유어.

**출처 표시**
- **ⓦ** = `ws_auto` 코퍼스에서 JP↔KO 정렬로 실제 확인된 표기
- **○** = 표기 방침(음차) 적용, 코퍼스와 상충 없음
- **⚠** = 재검토 후보(사용자 확정 필요)

## 부록 B. 조사·표기 규칙 (룰북 문체)

- 소울/스톡 등 **ㄹ·ㄱ 받침 외래어의 조사**는 발음 기준으로 (소울**로**, 스톡**으로**).
- 아이콘 심볼은 PDF에서 글리프가 빠지므로 룰북에서는 `[카운터 아이콘]` `[클록 아이콘]` `[소울 아이콘]` 형태로 표기.
- 원문 강조 마커 `「X … X」`(볼드/하이라이트 아티팩트)는 일반 참조 「…」로 정리.
- 상호 참조 「후술 "○○" 참조」의 대상 명칭은 실제 장/절 제목과 일치시킨다.

## 재검토 후보 (⚠) — ws_auto 기준 채택 확정, 필요 시 재조정
> 사용자 지시("최대한 ws_auto 기준")에 따라 아래는 **ws_auto 표기로 확정**함. 기록만 남김.
1. **助太刀 → 조태도** (코퍼스 표준 채택)
2. **投了 → 기권** (파일럿 채택)
3. **트리거 아이콘 명칭** — リターン→바운스, プール→스톡, カムバック→살베지 (ws_auto 내부 명칭 채택)
4. **オーナー/マスター → 오너/마스터**, **ターン回数制限 → 턴 횟수 제한** — 코퍼스에 룰 용례 없어 방침 적용(순수 음차/의역)
